__proto__和prototype

Sep 20, 2017 | javascript

使用__proto__属性:访问实例的原型

prototype属性:指向函数的原型对象

记住只有函数才有prototype属性,这个属性值为一个object对象

实例对象时没有这个属性的，实例对象通过__proto__这个内部属性（[[prototype]]）来串起一个原型链的，通过这个原型链可以查找属性，方法

通过new操作符初始化一个函数对象的时候就会构建出一个实例对象，函数对象的prototype属性指向的对象就是这个实例对象的原型对象，也就是__proto__指向的对象

构造函数是function,实例是Object,不是function

但是一般只有当把一个函数当构造函数使用时，我们才使用prototype

注：proto前后是两条下划线，不是一条
