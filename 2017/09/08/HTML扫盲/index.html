HTML扫盲

Sep 8, 2017 | HTML

注：本文记录了我的一些记得不清的知识点

标签：

meta:

<meta>元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词。

link:

<link>标签定义文档与外部资源的关系。

<link>标签最常见的用途是链接样式表。

base:

标签为页面上的所有链接规定默认地址或默认目标。

注： 标签必须位于 head 元素内部。

label：

label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上。

意思就是：当点击label标签时，浏览器光标会转到控件的id值和该label的for属性值相同的控件上。

注释：”for” 属性可把 label 绑定到另外一个元素。请把 “for” 属性的值设置为相关元素的 id 属性的值。

textarea:

<textarea> 标签定义多行的文本输入控件。

注：在文本输入区内的文本行间，用 “%OD%OA” （回车/换行）进行分隔。

script:

<script> 标签用于定义客户端脚本，比如 JavaScript。

script 元素既可以包含脚本语句，也可以通过 src 属性指向外部脚本文件。

必需的 type 属性规定脚本的 MIME 类型。

iframe:

iframe 元素会创建包含另外一个文档的内联框架（即行内框架）。

提示：您可以把需要的文本放置在 <iframe> 和 </iframe> 之间，这样就可以应对无法理解 iframe 的浏览器。

select:下拉表

select 元素可创建单选或多选菜单。(multiple属性可选择多个选项)

元素中的

自定义列表：

列表除了有序列表ol li，无序列表ul li,还有自定义列表dl dt dd

<dl> 标签定义了定义列表（definition list）。

<dl> 标签用于结合 <dt> （定义列表中的项目）和 <dd> （描述列表中的项目）。

图像标签：

除了img,还有map

map标签：定义一个客户端图像映射。图像映射（image-map）指带有可点击区域的一幅图像。

area标签：<area>标签定义图像映射中的区域（注：图像映射指得是带有可点击区域的图像）。

注：area 元素总是嵌套在 标签中。

注： 中的 usemap 属性可引用 中的 id 或 name 属性（由浏览器决定），所以我们需要同时向 添加 id 和 name 两个属性。

属性：

target：target 属性规定在何处打开 action URL。

alt:必需的 alt 属性规定图像的替代文本，如果该图像无法显示。

当用户无法查看图像时（可能由于网速太慢、错误的 src 属性、或者用户使用的是屏幕阅读器），alt 属性为图像提供了替代的文本。
